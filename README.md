# 🌐 DevOps Portfolio Website

A clean, modern, multi‑page DevOps portfolio built with **HTML**, **CSS**, and **JavaScript**.  
This project is designed to be simple to customize, easy to deploy, and visually polished — perfect for showcasing your DevOps, SRE, or Platform Engineering experience.

---

## ✨ Features

### 🔹 Multi‑page layout
Includes the following pages:
- **Home** (`index.html`)
- **Skills** (`skills.html`)
- **Experience** (`experience.html`)
- **Projects** (`projects.html`)
- **Blog** (`blog.html`)

### 🔹 Dark / Light theme toggle
- Smooth theme switching  
- Saves user preference using `localStorage`  
- Works across all pages  

### 🔹 Modern UI
- Clean typography  
- Soft shadows  
- Gradient hero text  
- Responsive grid layout  
- Beautiful dark theme with a polished light mode  

### 🔹 Easy to customize
You can edit:
- Name  
- Years of experience  
- Skills  
- Projects  
- Blog posts  
- Contact links  
- Colors and styles
